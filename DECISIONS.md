# DECISIONS.md — 参道 v1 実装判断ログ

- **スタック**: 単一静的 HTML（React/Vite なし）。Ataraxia Works 既存アプリと統一、デプロイ即可。
- **寺名データ**: `sando.html`（参照実装）が未提供のため CLAUDE.md 仕様のみから構築。寺名は公開されている四国八十八ヶ所霊場会の公式記録に基づく実名を収録（捏造ゼロ）。HONZON は空欄とし人間が公式情報から流し込む形を維持。
- **「一呼吸、おく」**: タイマーなし・ユーザーが「おく」を押して進む。自動進行はしない（ユーザーの主体性を守る）。
- **結願演出**: 静かに一文のみ。大きなアニメ・音・達成ポップアップなし。高野山奥の院への案内テキストを示す。
- **色**: 文字を薄くしない（§2-4）。`--text` = #e8e4dc（暖白）、最低コントラスト比 7:1 以上を保証。
- **参拝取り消し**: 納経帳から各札所の詳細に入り「取り消す」ボタンで削除可能。
